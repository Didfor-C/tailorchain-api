🐾 NF Cat API 문서 🐾

백엔드 API를 React + TypeScript 프론트엔드에서 쉽게 사용할 수 있도록 작성된 설명서입니다!

소개

이 문서는 NF Cat API의 엔드포인트를 설명하고, 프론트엔드에서 바로 사용할 수 있는 코드를 제공합니다. 복사하기 버튼을 눌러 쉽게 활용하세요!

엔드포인트

1. 회원가입

URL: POST /user/signup

설명: 새로운 유저 계정을 생성합니다.

Request Body:

{
  "email": "string",
  "password": "string"
}

Response: 생성된 유저 정보와 토큰이 포함된 JSON 객체

2. 로그인

URL: POST /user/login

설명: 유저를 인증하고 토큰을 반환합니다.

Request Body:

{
  "email": "string",
  "password": "string"
}

Response: 유저 정보와 토큰이 포함된 JSON 객체

3. 유저 정보 조회

URL: GET /user/:userId

설명: 특정 유저의 정보를 가져옵니다.

Headers:

Authorization: Bearer [token]

Response: 유저 정보가 포함된 JSON 객체

4. 유저 정보 수정

URL: PUT /user/:userId

설명: 특정 유저의 정보를 수정합니다.

Headers:

Authorization: Bearer [token]

Request Body:

{
  "userName": "string",
  "profileImage": "string",
  "darkMode": true,
  "pushNotification": true
}

Response: 수정된 유저 정보가 포함된 JSON 객체

팁

  • 모든 인증이 필요한 요청에는 Authorization 헤더를 포함하세요.
  • 프론트엔드에서 유저 입력 값을 검증하여 불필요한 에러를 방지하세요.
  • 토큰은 로컬 스토리지 또는 쿠키에 안전하게 저장하세요.

✨ 프론트엔드에게 제공되는 원래 귀엽게 태어난 API 문서 ✨
